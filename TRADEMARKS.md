**Last Updated**: January 2026

LlamaBrain™ is MIT-licensed open-source software. However, the name "LlamaBrain" and any associated logos or marks are trademarks and are subject to separate usage rules from the software license.

**Important**: The MIT License grants you rights to use, modify, and distribute the software, but it does not grant you rights to use the LlamaBrain trademark, name, or logo.

## Trademark Owner

## Allowed Uses (No Permission Required)

You may use the LlamaBrain name in the following ways without explicit permission:

### Nominative Use

You may use "LlamaBrain" to refer to the software itself:

- ✅ "Works with LlamaBrain"
- ✅ "Plugin for LlamaBrain"
- ✅ "Compatible with LlamaBrain"
- ✅ "Built using LlamaBrain"
- ✅ "LlamaBrain integration"
- ✅ "LlamaBrain fork" (see Fork Branding section below)

### Documentation and Technical References

- ✅ Referencing LlamaBrain in technical documentation
- ✅ Citing LlamaBrain in academic papers or research
- ✅ Listing LlamaBrain as a dependency or integration
- ✅ Using "LlamaBrain" in code comments or configuration files

### Use as an Adjective

Always use "LlamaBrain" as an adjective, not as a noun or verb:

- ✅ "LlamaBrain software" or "LlamaBrain library"
- ❌ "LlamaBrain" (as a standalone noun)
- ❌ "LlamaBraining" (as a verb)

## Uses Requiring Permission

You must obtain explicit written permission before:

### Commercial Use of Logo

- Using the LlamaBrain logo in marketing materials
- Using the logo on product packaging
- Using the logo in advertisements or promotional content
- Using the logo in presentations or conference materials

### Product/Company Naming

- Naming a product or company that incorporates "LlamaBrain"
- Creating a derivative product name that could cause confusion
- Using "LlamaBrain" as part of a domain name for commercial purposes

### Events and Conferences

- Using "LlamaBrain" in event names (e.g., "LlamaBrain Conference")
- Using the logo on event materials
- Creating "official" or "sponsored" LlamaBrain events

### Endorsement or Sponsorship

- Implying endorsement by or affiliation with LlamaBrain
- Using the mark in a way that suggests official partnership
- Creating "certified" or "approved" programs using the mark

## Prohibited Uses

You may not:

### Imply Endorsement

- ❌ Use the mark in a way that implies LlamaBrain endorses your product or service
- ❌ Create "LlamaBrain Certified" or "LlamaBrain Approved" programs without permission
- ❌ Use the mark to suggest official partnership or sponsorship

### Confusing Branding

- ❌ Use similar branding, colors, or design that could cause confusion
- ❌ Create logos or marks that are confusingly similar to LlamaBrain
- ❌ Use the mark in a way that dilutes its distinctiveness

### Fork Branding

- ❌ Ship forks under the "LlamaBrain" name or logo
- ❌ Use "LlamaBrain" as the primary name for a fork
- ❌ Create forks that could be mistaken for official LlamaBrain releases

**Fork Naming Requirement**: Forks must use a distinct name. Acceptable examples:
- ✅ "XBrain (a fork of LlamaBrain)"
- ✅ "LlamaBrain-Fork" (with clear distinction)
- ❌ "LlamaBrain" (as the primary name)

## Trademark Symbol Usage

When using "LlamaBrain" in prominent contexts (e.g., documentation, marketing), use the ™ symbol on first mention:

- ✅ "LlamaBrain™ is a deterministic state management library..."
- ✅ "The LlamaBrain™ project provides..."

After first mention, the symbol is optional but may be included for consistency.

**Note**: The ™ symbol indicates an unregistered trademark. If the mark becomes registered in the future, we will update this policy to use the ® symbol.
