## Attribution Requirements

If you distribute LlamaBrain or create derivative works:

1. **Include this file**: Maintain this THIRD_PARTY_NOTICES.md file in your distribution
2. **Preserve license notices**: Keep all copyright and license notices intact
3. **Comply with licenses**: Ensure your use complies with each third-party license
4. **Check redistribution rights**: Some assets may not be redistributable

## License Compatibility

All core dependencies use permissive licenses (MIT) that are compatible with LlamaBrain's MIT License. Unity-specific dependencies may have different terms; review each package's license before redistribution.

## Reporting Missing Attribution

If you notice missing attribution for any third-party code or assets, please:
1. Open a GitHub issue with the `attribution` label
2. Provide details about the code/asset and its source
3. Include license information if available
